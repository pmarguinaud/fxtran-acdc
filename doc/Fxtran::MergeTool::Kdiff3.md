# NAME

[Fxtran::MergeTool::Kdiff3](../lib/Fxtran/MergeTool/Kdiff3.pm)

# DESCRIPTION

Backend for the kdiff3 merge tool. Implements the `merge` method from
[Fxtran::MergeTool](Fxtran%3A%3AMergeTool.md) by invoking the `kdiff3` external command to perform
a three-way merge of base, local, and remote files, writing the result to
the merged output file.
