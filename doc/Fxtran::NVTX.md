# NAME

[Fxtran::NVTX](../lib/Fxtran/NVTX.pm)

# DESCRIPTION

This module provides the `drHookToNVTX` function, whose purpose is
to convert `DR_HOOK` calls to `NVTX` calls. 

## ifdef

Wrap a statement with `#ifdef` / `#endif` preprocessor guards for the given macro.

## drHookToNVTX

Convert `DR_HOOK` calls in a program unit to their `NVTX` equivalents.
