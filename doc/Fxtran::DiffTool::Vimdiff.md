# NAME

[Fxtran::DiffTool::Vimdiff](../lib/Fxtran/DiffTool/Vimdiff.pm)

# DESCRIPTION

Backend for the vimdiff diff tool. Implements the `diff` method from
[Fxtran::DiffTool](Fxtran%3A%3ADiffTool.md) by invoking the `vimdiff` external command to display
a side-by-side comparison of two files inside Vim.
