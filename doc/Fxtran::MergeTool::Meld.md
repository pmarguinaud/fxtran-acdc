# NAME

[Fxtran::MergeTool::Meld](../lib/Fxtran/MergeTool/Meld.pm)

# DESCRIPTION

Backend for the meld merge tool. Implements the `merge` method from
[Fxtran::MergeTool](Fxtran%3A%3AMergeTool.md) by invoking the `meld` external command to perform
a three-way merge of base, local, and remote files, writing the result to
the merged output file.
